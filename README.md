# 自主ゼミ資料共有リポジトリ

このリポジトリは自主ゼミの資料管理用です。
運用をスムーズにするため、以下のルールに従ってファイルを管理してください。

## 📁 ファイル命名規則

### 1. ゼミのスライド資料
- **形式**: `kaiseki[回数].tex`
- **例**: `kaiseki1.tex`, `kaiseki2.tex`
- 第何回目の資料かがわかるように、数字を入れて保存してください。

### 2. スライドの書き方
`sample.tex`にbeamerの基本機能と例をまとめています。
参考にしてください。

### 3. その他の資料
- **命名**: 内容にちなんだ名前にすること。
- **制約**: **ひらがな・漢字は使用禁止**です（英数字・ハイフン・アンダースコアのみ）。
- **例**: `derivation-logic.pdf`, `exercise_ref.png`
